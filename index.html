cybercitizens.io

html, scss, js, debian, nginx, certbot

CyberCitizens is a NFT project based on the ERGO blockchain. Being lovers of pixel art and seeing the popularity of their first NFT drop, CyberCitizens will be expanding into the metaverse come 2022. Entry within CyberCity is reserved for CyberCitizen holders. Within the CyberCity expect to find a variety of games to take part in, chances to socialize with others and much more!

YAYO NFT

html, css, js, web3.js, ethers.js, gimp

YAYO NFT is an NFT collection based on the MATIC network using the yayo ERC20 yayo token. The site allows user to purchase YAYO NFTs using the yayo token, written using web3.js and ethers.js to interact with MetaMask to faciliate transactions. This project is fully responsive and usable on mobile, tablet, and computer. Currently apple devices are unsupported.
