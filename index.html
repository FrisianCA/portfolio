Projects

ERGOFund

react, wasm, mui, yoroi, nautilus

ERGOFund is a tool that seeks to provide an all in one payment solution for dApps on ERGO. ERGOFund will be packaged and available to be inserted into dapps, allowing customers or supporters to easily send transactions using the Nautilus wallet. ERGOFund hopes to be a major tool in the ERGO ecosystem, giving developers more time to focus on their platform, and not on the implementation of transaction processing. In the future, we hope to offer solutions for new and emerging wallets and dApp connectors, to ensure the widest range of support for the ERGO ecosystem as possible.

YAYO NFT

html, css, js, web3.js, ethers.js, gimp

YAYO NFT is an NFT collection based on the MATIC network using the yayo ERC20 yayo token. The site allows user to purchase YAYO NFTs using the yayo token, written using web3.js and ethers.js to interact with MetaMask to faciliate transactions. This project is fully responsive and usable on mobile, tablet, and computer. Currently apple devices are unsupported.
